# M&A/투자유치 자문 업무 AI 자동화 시스템

## 문제

> M&A/투자유치 자문 회계사의 반복적인 리서치·커뮤니케이션 업무를 AI 스킬로 자동화하여, 고부가가치 딜 판단과 클라이언트 관계에 집중할 수 있게 한다.

- **현재 상태**: 딜 소싱 리서치 1건당 2~3시간 소요, 주간 파이프라인 정리 30~60분, 클라이언트 이메일 초안 건당 30분 — 주간 반복 업무만 10시간 이상
- **원하는 상태**: 회사명 하나 입력으로 딜 소싱 리포트 10분 이내 생성, 파이프라인 현황 자동 집계, 이메일 초안 즉시 생성 — Claude가 반복 업무를 처리하고 나는 의사결정에 집중
- **성공 기준**: 딜 리서치 소요 시간 80% 단축 (2~3시간 → 30분 이내), 주간 파이프라인 정리 자동화 (수동 0분)

## 변화 기록

- **Day 1**: "Claude Code가 뭔지 알겠다. 뭔가 만들 수 있을 것 같다" → Hook, Skill, MCP의 개념을 배우고 첫 커스텀 스킬 작성
- **Day 2**: "MCP로 외부 서비스를 연결할 수 있다" → Gmail·Google Calendar·Slack·GitHub을 Claude에 연결하고 `my-context-sync` 스킬로 정보를 자동 수집
- **Day 3**: "요구사항을 명확히 정의하는 것 자체가 기술이다" → `my-clarify` 스킬로 모호한 업무 요청을 구체적 스펙으로 변환하는 법 체득
- **가장 크게 달라진 점**: "Claude에게 시키는 것"에서 "Claude가 스스로 판단하고 실행하는 시스템을 설계하는 것"으로 관점이 바뀌었다. 도구 사용자에서 도구 설계자로.
